# Nexus Retail Sales & Supply Chain Optimization

## Project Overview
This portfolio project demonstrates end-to-end data capabilities as a Business Analyst. Using an e-commerce scenario ("Nexus Retail"), this project tackles profitability leakage, customer retention metrics, and supply chain operational bottlenecks using **Advanced Excel** and **Power BI**.

## Project Roadmap
- [x] **Phase 1:** Data Architecture & Relational Modeling (Excel) — *Current Stage*
- [ ] **Phase 2:** ETL & Data Cleansing (Power Query)
- [ ] **Phase 3:** Advanced Metrics & Data Modeling (DAX / Power BI)
- [ ] **Phase 4:** Interactive Dashboard & Business Insights

## Current Progress
Successfully built the core relational tables (`tbl_Products`, `tbl_Customers`, and `tbl_Sales`) and established dynamic price fetching utilizing structured table references and `XLOOKUP`.
